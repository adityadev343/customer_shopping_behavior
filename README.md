# Customer Shopping Behavior Analytics Dashboard

Streamlit dashboard for analyzing customer shopping data with AI-powered SQL assistant (Gemini).

## Features
- 8 tabs: Revenue, Customer, Product, Discount & Promo, Shipping, Geographic, Payment, Advanced
- Dynamic filters (gender, age, category, season, subscription, discount, promo, shipping, location)
- Interactive Plotly charts, KPIs, customer segmentation, product affinity
- AI SQL Assistant: plain English → SQL → business insights

## Local Setup
```bash
git clone https://github.com/adityadev343/customer_shopping_behavior.git
cd customer_shopping_behavior
pip install -r requirements.txt
streamlit run app.py
